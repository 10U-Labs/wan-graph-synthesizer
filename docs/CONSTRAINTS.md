# Project Constraints

This project must be built using free and open-source tools only. Any framework,
library, service, or supporting utility should be usable without proprietary
licensing requirements or paid-only access.

The application must be self-hosted. It should not depend on managed SaaS
platforms for core functionality, and all required services should be deployable
and operable by the project owner.

All functionality must be exposed through APIs. The web interface should consume
the same API surface available to other clients, with no hidden application logic
that is only accessible through the UI.

The primary user experience must be delivered as a web application. The system
should be accessible through a browser and designed around web-based workflows.
